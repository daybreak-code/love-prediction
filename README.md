# Love Prediction

一个基于AI的情感分析和预测平台，帮助用户找到真正适合的伴侣。

## 技术栈

- Next.js 14
- React 18
- Framer Motion
- React Icons
- CSS Modules

## 功能特点

- 响应式设计
- 现代化UI/UX
- 流畅的动画效果
- 移动端优化

## 开始使用

1. 安装依赖：

```bash
npm install
```

2. 运行开发服务器：

```bash
npm run dev
```

3. 构建生产版本：

```bash
npm run build
```

4. 运行生产版本：

```bash
npm start
```
